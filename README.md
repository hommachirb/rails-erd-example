Ruby on Rails ERD Example
=========================

## Installation

```
$ git clone https://github.com/hommachirb/rails-erd-example.git
$ cd rails-erd-example/
$ bundle
$ rake db:setup
$ rails server
```

## Use

```
$ open http://localhost:3000/rails/draw/erd/
```
